## 项目说明

    本项目开发的目的有四：

        1、作为一个使用SD.Framework框架开发的项目样板；
        
        2、探索基于ASP.NET MVC的前端架构；

        3、探索基于WPF MVVM的前端架构；

        4、做一套统一身份认证系统；


    注意事项：

        请先修改服务端Host项目配置文件中连接字符串，然后运行，数据库会自动生成

        先运行服务端，再运行客户端

        如果服务端运行失败，请检查是不是没装Redis（附件中有）

        默认账号：admin，密码：888888
